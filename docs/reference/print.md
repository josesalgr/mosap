# Print

Displays information about an object.

## Usage

``` r
# S3 method for class 'Data'
print(x, ...)

# S3 method for class 'Solution'
print(x, ...)

# S3 method for class 'MOProblem'
print(x, ...)
```

## Arguments

- x:

  Any object.

- ...:

  Not used.

## Value

None.
