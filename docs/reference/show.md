# Show

Display information about an object.

## Usage

``` r
# S4 method for class 'Data'
show(x)

# S4 method for class 'OptimizationProblem'
show(x)

# S4 method for class 'Portfolio'
show(x)
```

## Arguments

- x:

  Any object.
